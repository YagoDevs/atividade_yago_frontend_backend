# Atividade Ponderada - Yago Phellipe

Este repositório contém um projeto de site de música com o backend desenvolvido em Java Spring Boot e o frontend em React TypeScript. No backend, foi implementada uma arquitetura simples, incluindo entidades, repositórios, serviços e controladores. No frontend, foi utilizado o padrão de desenvolvimento do React.

Na página inicial do site, é possível realizar diferentes tipos de requisições, como GET para obter informações sobre as músicas, POST para adicionar novas músicas, UPDATE para modificar informações das músicas e DELETE para remover músicas. Além disso, é possível clicar no nome de uma música para ser redirecionado ao seu vídeo correspondente no YouTube.

O banco de dados utilizado foi o PostgreSQL, hospedado no serviço RDS da Amazon Web Services (AWS). Tanto o frontend quanto o backend foram hospedados em instâncias EC2 da AWS, com o frontend executando em uma máquina Ubuntu e o backend em uma máquina com o terminal AWS.

Este projeto foi desenvolvido como parte de uma atividade acadêmica e tem como objetivo demonstrar o uso de tecnologias como Spring Boot, React e PostgreSQL para criar uma aplicação web moderna e funcional.
